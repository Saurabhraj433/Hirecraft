# Hirecraft

Welcome to **Hirecraft** - Your go-to platform for finding the best job opportunities and connecting with top talent!

## 🌐 Live Link
**Check out our platform:** [https://hirecraftt.netlify.app/](https://hirecraftt.netlify.app/)

## 📋 About
Hirecraft is a job matching platform that connects job seekers with employers. Whether you're looking for your next career move or searching for the perfect candidate, Hirecraft makes it simple and easy.

## ✨ Features
- **Job Listings** - Browse through hundreds of job opportunities
- **Easy Application** - Apply to jobs in just a few clicks
- **Profile Building** - Create your professional profile
- **Employer Dashboard** - Post jobs and manage applications
- **Search & Filter** - Find jobs that match your skills
- **Direct Messaging** - Connect with employers and candidates

## 🚀 How to Use
1. Visit [https://hirecraftt.netlify.app/](https://hirecraftt.netlify.app/)
2. Sign up or log in to your account
3. Browse job listings or post a job
4. Apply for jobs or review applications
5. Connect and start your journey!

## 💻 Technology Stack
- **Frontend:** React, HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Deployment:** Netlify

## 🛠️ Setup & Installation (For Developers)

### Prerequisites
- Node.js installed on your system
- MongoDB running locally or connected to Atlas

### Steps to Run Locally
1. Clone the repository
   ```bash
   git clone https://github.com/Saurabhraj433/Hirecraft.git
   cd Hirecraft
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Set up environment variables (create `.env` file)
   ```
   MONGODB_URI=your_mongodb_connection_string
   PORT=5000
   ```

4. Start the server
   ```bash
   npm start
   ```

5. Open your browser and visit `http://localhost:3000`
